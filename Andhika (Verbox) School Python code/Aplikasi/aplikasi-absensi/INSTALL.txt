# PANDUAN INSTALASI DAN PENGGUNAAN
# Aplikasi Absensi Sekolah

## CARA INSTALASI

### Untuk Windows:

1. Pastikan Python sudah terinstall (minimal Python 3.8)
   - Download Python dari: https://www.python.org/downloads/
   - Centang "Add Python to PATH" saat instalasi

2. Extract folder aplikasi ke lokasi yang diinginkan

3. Buka Command Prompt (CMD) atau PowerShell

4. Masuk ke folder aplikasi:
   ```
   cd path\ke\folder\aplikasi
   ```

5. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

6. Jalankan aplikasi:
   - Double click file `run.bat`, atau
   - Ketik di CMD: `python app.py`

7. Buka browser dan akses: http://localhost:5000

### Untuk Linux/Mac:

1. Pastikan Python3 sudah terinstall

2. Extract folder aplikasi

3. Buka Terminal

4. Masuk ke folder aplikasi:
   ```
   cd /path/to/aplikasi
   ```

5. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
   
   Atau jika menggunakan pip3:
   ```
   pip3 install -r requirements.txt
   ```

6. Jalankan aplikasi:
   ```
   bash run.sh
   ```
   
   Atau:
   ```
   python3 app.py
   ```

7. Buka browser dan akses: http://localhost:5000

## CARA PENGGUNAAN

### ADMIN

1. LOGIN sebagai admin

2. KELOLA KELAS:
   - Klik menu "Kelola Kelas"
   - Klik tombol "Tambah Kelas Baru"
   - Masukkan nama kelas (contoh: Kelas 7A)
   - Masukkan daftar siswa (satu nama per baris)
   - Klik "Simpan Kelas"

3. KELOLA USER (GURU):
   - Klik menu "Kelola User"
   - Klik tombol "Tambah User Baru"
   - Isi data guru:
     * Username (contoh: guru2)
     * Nama lengkap (contoh: Bu Siti)
     * Password
     * Pilih role: Guru atau Admin
   - Klik "Simpan User"

4. LIHAT LAPORAN:
   - Klik menu "Laporan"
   - Lihat riwayat semua absensi
   - Klik "Detail" untuk melihat detail per absensi
   - Bisa di-print langsung

## TROUBLESHOOTING

### Error: Module 'flask' not found
Solusi: Install Flask dengan perintah:
```
pip install Flask
```

### Error: Port 5000 already in use
Solusi: 
1. Tutup aplikasi lain yang menggunakan port 5000
2. Atau edit file app.py, ganti port 5000 dengan port lain (misal 8000)

### Lupa password
Solusi:
1. Hapus folder `data`
2. Restart aplikasi
3. Akun default akan dibuat ulang

### Aplikasi tidak mau jalan
Solusi:
1. Pastikan Python versi 3.8 atau lebih baru
2. Pastikan Flask sudah terinstall
3. Cek error message di terminal/CMD
4. Restart aplikasi
